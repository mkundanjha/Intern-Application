This is python program for Socket Programming.

In this programs if a user at client side enter correct UserId and Password then they are logged into the server.
The detail of the user i.e. UserId and Password is stored in the server.

Step for creating the connection.
-Firt a server is needed to be established so first run the 'server.py'.
-Second run the 'client.py' for client interface. Here you can enter the UserId and Password.
