This is an Amazon Scrapper script designed using python.It is a command line interface.

The user is required to choose between 'entering a url' or the 'ASIN id' to make a search.
Entering either of the above with fetch the details of the products to the user.
The details include:
     - Name
     - Price of product
     - Ratings of the Product
     - Summary or Specification of the product

#Note:
     - This script works only for 'www.amazon.in'- especially while using the ASIN id.
     - ASIN id is unique id that Amazon assigns to each of its products.
